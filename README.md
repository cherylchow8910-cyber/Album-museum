# Album Museum 🎵

A beautiful digital art gallery for music albums built with Next.js 14, TypeScript, Tailwind CSS, and Framer Motion.

## Quick Start

```bash
git clone https://github.com/cherylchow8910-cyber/album-museum.git
cd album-museum
npm install
npm run dev
```

Open http://localhost:3000

## Features

- 🎨 Dark Museum Theme
- 🖼️ Gallery Experience
- 🎵 Music Integration (Spotify/YouTube)
- ❤️ Curated Collections
- 🔍 Smart Search
- 📱 Responsive Design
- ✨ Smooth Animations

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
